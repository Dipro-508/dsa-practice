# DSA practice

Contains:
- factorial (recursive) in C
- sum of arithmetic progression (AP) in C
- fibonacci (full recursion) in C
## Run & Test

Windows (PowerShell)
1. Build: `.\build.ps1`
2. Run tests: `.\run_tests.ps1`
3. View outputs: `Get-Content .\tests\outputs\factorial_output.txt` (or open in Explorer)

Linux/macOS
1. Build: `make`
2. Run tests (example): `./factorial/factorial < tests/inputs/factorial_input.txt`
